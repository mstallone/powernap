# Changelog

This project follows semantic versioning after the first public release.

## 0.1.0 - Unreleased

- Initial local-only PowerNAP implementation for Codex, Claude Code, and generic commands.
- Per-user `powernapd` and `powernap-watchdog` LaunchAgents.
- Active-turn power leases with idle sleep assertions and clamshell override cleanup.
- Codex hook installer and Claude per-run settings overlay.
- Companion menu bar app with sleep state icon, active protected thread count, detailed status menu, restore, refresh, logs, and quit actions.
- Setup, troubleshooting, uninstall, contribution, and security documentation.
- Release metadata, CI, contribution guidance, and security policy.
